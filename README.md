## Sistema de Clínica de Exames de Sangue

Este projeto é um sistema simples desenvolvido em Python para simular o funcionamento de uma clínica de exames de sangue. Ele permite gerenciar pacientes, exames e atendimentos de forma prática.

## Funcionalidades

- Cadastro de pacientes
- Agendamento de exames
- Listagem de exames disponíveis
- Registro de atendimentos
- Validação de dados (ex: entradas numéricas)
- Interface via terminal (CLI)

## Tecnologias utilizadas

- Python 3
- Programação estruturada
- Entrada e saída de dados via terminal

## Como executar o projeto
1. Clone o repositório:
git clone https://github.com/Gustavo-Lima7/projeto_clinica_de_exame_de_sangue.git

2. Acesse a pasta do projeto:
cd projeto_clinica_de_exame_de_sangue

3. Execute o arquivo
python teste_projeto.py
